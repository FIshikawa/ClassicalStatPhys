# ClassicalStatPhys

Project related to classical Hamiltonian dynamics (e.g. FPU model, Toda Lattice, Harmonic Chain).


## How to compile
```
$ mkdir build & cd build
$ cmake -DCONFIG=gcc_or_intel ../
$ make 
```
